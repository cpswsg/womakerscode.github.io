wo ma kers code

Empoderar é o primeiro passo para novas revoluções

O WoMakersCode é um projeto livre, criado e mantido por voluntários ligados à comunidades open-source, que tem como objetivo incentivar o aprendizado e a participação de mulheres em áreas como robótica, desenvolvimento e qualidade de software. São desenvolvidas oficinas com base teórica e prática, onde o aprendizado colaborativo e compartilhamento de experiências, são parte fundamental.

Javascript

Nesta edição, teremos um dia dedicado ao aprendizado de Javascript! JavaScript é uma linguagem de programação em extrema ascensão, que permite desenvolver aplicações web, mobile, desktop e até criar interações com hardware. Para aproveitar a oficina, você precisa ter noções de HTML e CSS.

Crie ou organize uma nova oficina

Quer compartilhar o que você sabe? Será um prazer contar com sua ajuda para criar uma oficina e junt@s aprendermoz algo novo!
